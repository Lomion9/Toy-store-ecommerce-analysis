# 🧸 The Toy Store — E-Commerce BI Dashboard

An end-to-end Power BI dashboard analyzing sales performance, 
product profitability, and traffic sources for a toy e-commerce 
store using Star Schema modeling and DAX.

📄 [Türkçe README için tıklayın](README_TR.md)

---

## 📊 Dashboard Pages

**Page 1 — Executive Summary**
Overview of key business metrics including Revenue, Orders, 
Gross Profit, Gross Margin, AOV, and Conversion Rate with 
year-over-year trends and monthly KPI tracking.

**Page 2 — Product Analysis**
Product-level breakdown of revenue, gross profit, refund rates, 
and margin performance. Identifies highest margin and most 
returned products.

**Page 3 — Traffic & Campaign Analysis**
Device type performance, session volume, and campaign-level 
conversion rates. Key insight: Brand Search drives the highest 
conversion rate despite lower order volume.

---

## 🔍 Key Insights

- The Original Mr. Fuzzy generates the highest revenue but also 
the highest refund rate
- The Birthday Sugar Panda delivers the strongest gross margin
- Brand Search campaigns convert at 7.8% — highest among all 
channels
- Desktop dominates both sessions and conversions over mobile

## 🛠️ Tools & Techniques

- **Power BI** — Dashboard design and visualization
- **DAX** — KPI measures, MoM calculations, time intelligence
- **Power Query / M** — Data cleaning and calendar table
- **Data Modeling** — Star Schema with 6 tables
